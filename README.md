# 神秘学 仪式挎包

这是一个非官方移植模组，将 [Occultism](https://www.curseforge.com/minecraft/mc-mods/occultism) 1.21.1版本中的 仪式挎包
，移植到了1.20.1版本。

## 整合包作者须知

如果您在整合包中使用了此模组，并且新增的仪式包含了新的方块，请务必将这些新方块添加到 `occultism_rs:pentacle_materials`
方块标签中，以确保挎包的正常运作。

# Occultism Ritual Satchel

This is an unofficial port mod that adds the "Ritual Satchel"
from [Occultism](https://www.curseforge.com/minecraft/mc-mods/occultism) 1.21.1 to version 1.20.1.

## For Modpack Authors

If you are using this mod in your modpack and have added new rituals that involve new blocks, please ensure you add
these new blocks to the `occultism_rs:pentacle_materialsblock` tag. This is necessary for the satchel to function
correctly.
